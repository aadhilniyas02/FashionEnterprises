# Fashion Enterprises


This is a premium, modern landing page for a garment/apparel brand, built with:
- React (Vite)
- Tailwind CSS
- Framer Motion (for animations)
- Lucide React (for icons)

## How to Run

1.  **Install Dependencies** (if not already):
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:5173](http://localhost:5173) in your browser.

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure

- `src/LandingPage.jsx`: The main single-page component containing all sections.
- `src/App.jsx`: Entry point rendering the LandingPage.
- `src/index.css`: Global styles and Tailwind directives.
- `tailwind.config.js`: Tailwind configuration (colors, fonts).

## Sections
- Hero with parallax background
- About Us with feature cards
- History timeline
- Services grid
- Process stepper
- Why Choose Us (split layout)
- Gallery (Masonry style)
- Testimonials
- FAQ Accordion
- Contact Form
